# 前端页面开发
### 本地调试
- python 3.5
- pip install flask
- python test.py
### 页面逻辑
- index.html
introduce页面，作为项目介绍等等等等
- search.html
dataset页面，可以通过物种和性状进行搜索，搜索跳转到summary页面，进行搜索结果的集中展示
- summary.html
ppt2所示页面，进行搜索结果的展示，点击第一列进入details页面，对应ppt3-7
- details.html
ppt3-7页面展示，将tab选择框放到了左侧
